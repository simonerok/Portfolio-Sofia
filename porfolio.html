Designprocess

Choosing a font

When working with fonts i use "TypeScale.com" for testing sizes, families, and pairings. Font hierarchy is key for a website, and a have learned that a simple sans-serif display font works well and that it's best to stick with two font types and experiment with boldness and italics etc.

Image from Typescale.com

Choosing colors

Working with colors a great tool i like to use is "Huemint.com". With this program, you can experiment with different color combinations and generate color themes to visualize how they will appear on a website.

Image from Huemint.com

Style tiles and mockups

I believe that creating wireframes, style tiles, and mockups prior to coding can accelerate the website designprocess while also ensuring proper layout and that everything is well organized.
